🤖 Rule-Based Chatbot
A simple command-line chatbot built in Python that responds to user inputs using predefined rules, pattern matching with regular expressions, and keyword detection — no external NLP libraries required.

📌 Project Overview
This project was built as part of a CodSoft internship task to demonstrate the fundamentals of Natural Language Processing (NLP) and conversation flow using rule-based logic.

✨ Features

Greetings & farewells
Tells current time and date
Answers identity questions
Tells jokes
Basic arithmetic calculator
Weather info redirect
Mood detection
Help menu
Randomized responses for natural feel


🛠️ Technologies Used

Python 3
re module — regular expression pattern matching
datetime module — real-time date and time
random module — varied responses


🚀 How to Run
bashpython Chatbot.py
```

No installations needed. Just Python 3!
```
