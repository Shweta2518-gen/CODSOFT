# Face Detection using Python

## Project Description

This project implements a simple **Face Detection system** using Python.
The program detects human faces from an input image and draws rectangles around the detected faces.

## Technologies Used

* Python
* OpenCV (Computer Vision Library)
* Haar Cascade Classifier

## Features

* Detect faces from an image
* Draw bounding boxes around detected faces
* Count the number of faces in the image
* Display the result in a window

## Project Structure

```
TASK_3/
│
├── face_detection.py     # Main Python script
├── person.jpg            # Input image
└── FACE.md             # Project documentation
```

## How to Run the Project

1. Install the required library:

```
pip install opencv-python
```

2. Place the image file (`person.jpg`) in the same folder as the Python script.

3. Run the program:

```
python face_detection.py
```

4. The program will:

* Detect faces in the image
* Display the image with rectangles around detected faces
* Print the number of faces detected in the terminal

## Output

The program opens a window showing the image with detected faces highlighted using rectangles.
